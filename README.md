## 🖥️💡 Projeto Web via Terminal

## 📖📖 Descrição:
Projeto criado totalmente utilizando comandos no terminal Ubuntu.

## 🤖🤖 Comandos utilizados:
- pwd - Mostra o caminho da pasta atual.
- ls - Lista arquivos e pastas.
- cd - Entra em pastas.
- mkdir -  Cria pastas.
- touch -  Cria arquivos.
- cp - Copia arquivos ou pastas.
- mv -  Move ou renomeia arquivos ou pastas.
- rm - Remove arquivos (cuidado!).
- nano/vim -  Editores de texto internos.


## 😣😣 Dificuldades
- Aprender a usar o Vim/Nano
- Memorizar comandos
